começo do projeto

para começar o projeto no VScode usa o "!"

pular linhas

para pular a linha você coloca o <br>

paragrafos

para colocar um paragrafo você precisa usar o letra "p"
EX:<p> aqui dentro é um paragrafo </p>

para colocar titulos

nos titulos você usar o <h> seguido de um numero de 1 a 6 que representa o nivel do seu titulo e fechar com </h>
EX:

<h1>aqui é o primeiro titulo</h1>


<h2>aqui é o primeiro titulo</h2>


<h3>aqui é o primeiro titulo</h3>


<h4>aqui é o primeiro titulo</h4>


etc

para colocar emoji

os emojis são colocados com &#x e depois com o codigo de cada emoji que acha no site esse aqui para ver

sobre links

para colocar um link você precisar usar a letra "a" ele ja ira gerar o resto do codigo em "href" vai ser colocado o link para qual quer ser direcionado apos isso ira ser colocado oque ira se escrever como hipertexto
EX: <a href="https://emojipedia.org/">aqui oque ira estar escrito </a>

apos colocar o link quando apertar ira trocar na mesma pagina para a qual o link leva para que abra em outra guia se utiliza o "targt='_blank'"
EX:<a href="https://emojipedia.org/" target="_blank" >aqui oque ira estar escrito </a>

sobre imagens

para colocar imagens voce usa a tag "img" ela vai gerar o resto do codigo em "src" voce coloca o link da imagem que esta no seu computador ou a URL da imagem em "alt" voce coloca oque é a imagem
EX: <img src="URL" alt="imagem demonstração">

a imagem ocupa bastante espaço no site então lembre de redimencionar para ficar de um bom tamanho que não seja tão pesado

sobre favicon

para colocar favicon que seria a imagens do lado do titulo apenas colo que link e escolha a opção favicon em href coloque a imagem do favicon que você tiver

negrito e italico

para se colocar uma frase em negrito se usa <strong> para colocar uma frase em italico se usa <em>
EX:<strong>aqui esta em negrito</strong>
<em>aqui esta em italico</em>

outras formatações

você pode marcar o texto com a palavra mark por exemploapenas precisa usar o <mark>

você pode colocar um texto sublinhado por exemplo apenas precisa usar o <ins>

você pode colocar um texto riscadopor exemploapenas precisa usar o <del>

voce pode colocar colocar textos simplificados como blzapenas precisa usar o <abbr>

você pode colocar itens para cima ou para baixo pro exemplo x20 apenas precisa usar o <sup> e h20 apenas precisa usar o <sub>
