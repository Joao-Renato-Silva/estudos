tag para o texto

para se escrever o titulo se usa as tag <h1> que pode se ir de h1 ate h6, para começar o seu paragrafo se usa a tag <p> ela usa o tamanho de letra padrao e criando o paragrafo

tag para edição de frases

para se colocar a palavra em negritose utilizar o <strong>
para se colocar a frase em italico se utiliza a tag <em>
para se utilizar um marca texto se utliza a tag <mark>
para se usar a frase em underlinese utiliza a tag<ins>
para deixar a frase cancelada se usa a tag <del>
para se criar botoes se utiliza a tag <button>

falar sobre imagens

para se colcoar imagens vc precisa utilizar a tag <img src="" alt="">

sobre links

para se abrir um link pode se colocar na mesma pagina ou a redirecionar para outro para outra pagina para se fazer isso a imagem deve estar dentro de uma outra tag <a> apos isso se coloca <a href="" target="_blank"></a>

sobre favicon

para se colocar um favicon vc utiliza o <link rel="shortcut icon" href="favicon.ico" type="image/x-icon"> se prefere que a imagem esteja salva como .icon
